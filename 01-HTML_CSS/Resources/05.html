Let's learn other HTML tags shall we?

This is a paragraph. It can contain other tags that select some areas of the text to highlight, like bold, strong, emphasis, italics; you can also have a hyperlink in a paragraph. Note that the "strong", "emphasis", "italics", "bold" and "a" elements are all inline elements, while the p element is a block element.
